# Mysterious Vehicle Claim Key (MVCK)
## Thanks
Special thanks for Lu5ck author of Another Vehicle Claim System

## Info
Mysterious Vehicle Claim Key is an updated mod for works only in build 42, with some new features and updated ui and textures.

## Old Features
- Protection for Uninstalling Parts
- Protection for Taking Engine Parts
- Protection for Siphoning Gas
- Protection for Entering Vehicle
- Protection for Entering Driver Seat
- Protection for Opening (closed) trunk
- Protection for Attach or Detach Vehicle
- Protection for Inflate/Deflate Tires
- Allow/Disable share Vehicle with Faction on sandbox
- Allow/Disable share Vehicle with Refuge on sandbox

## New Features
- Added protection for Smash Car Windows.
- Merged mod for craft recipe with the base mod, now it can be disabled from sandbox options.
- Updated recipe with build 42 new items and mechanics.
- Updated textures by Trynnuchi
- Updated UI (WIP)
