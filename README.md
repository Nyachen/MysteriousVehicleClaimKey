# Mysterious Vehicle Claim Key (MVCK)
## Thanks
Special thanks for Lu5ck author of Another Vehicle Claim System [github: ](https://github.com/Lu5ck/AnotherVehicleClaimSystem)

## Info
Mysterious Vehicle Claim Key is an updated mod for works only in build 42, with some new features and updated ui and textures.

## New Features
- Added protection for car windows.
- Fused mod for craft recipe with the base mod, now it can be disabled from sandbox options.
- Updated recipe with build 42 new items and mechanics.
- Updated texture (Key thematics)
- Updated UI (WIP)
